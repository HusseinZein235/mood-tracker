# تطبيق moodtracker

تطبيق ويب لتتبع المزاج ومساعدة المستخدمين على تحسين صحتهم النفسية.

## الميزات

- تسجيل المستخدمين وتسجيل الدخول
- تتبع المزاج اليومي
- إدخال السبب والأنشطة للحالة المزاجية
- روبوت محادثة ذكي للدعم العاطفي
- أنشطة لتحسين المزاج (تنفس، استرخاء، تقنية بومودورو)
- لوحة تحكم المدير لمراقبة النظام

## متطلبات النظام

- PHP 7.0 أو أعلى
- MySQL 5.7 أو أعلى
- خادم ويب (Apache أو Nginx)

## التثبيت

1. قم بتنزيل أو استنساخ المستودع إلى مجلد خادم الويب الخاص بك (مثل `htdocs` إذا كنت تستخدم XAMPP)

2. تأكد من تثبيت PHP و MySQL وتشغيلهما

3. قم بتنفيذ `http://localhost/moodtracker/test.php` في متصفح الويب للتحقق من الإعداد وإنشاء قاعدة البيانات والجداول المطلوبة

4. سيقوم النظام تلقائيًا بإنشاء:
   - قاعدة بيانات `moodmate`
   - الجداول المطلوبة `users` و `mood_records` و `chat_history`
   - حساب المسؤول الافتراضي

## الوصول إلى التطبيق

- صفحة المستخدم: `http://localhost/moodtracker/index.html`
- لوحة تحكم المدير: `http://localhost/moodtracker/admin.php`

## بيانات اعتماد المسؤول

- اسم المستخدم: `admin`
- كلمة المرور: `12345678`

## بنية الملفات

- `index.html` - واجهة المستخدم الرئيسية
- `auth.php` - مصادقة المستخدم وتسجيل الدخول/الخروج
- `config.php` - اتصال قاعدة البيانات والإعداد
- `save_mood.php` - حفظ سجلات المزاج
- `get_moods.php` - استرجاع سجلات المزاج للمستخدم
- `admin.php` - لوحة تحكم المدير
- `test.php` - أداة لاختبار النظام
- `database.sql` - هيكل قاعدة البيانات

## تحقق من النظام

يمكنك استخدام `test.php` للتحقق من:
- اتصال قاعدة البيانات
- وجود الجداول المطلوبة
- حساب المدير
- إصدار PHP
- صلاحيات الملفات

## استكشاف الأخطاء وإصلاحها

إذا واجهت أي مشاكل:

1. تأكد من تشغيل خادم الويب وخادم MySQL
2. تحقق من أن لديك أذونات كافية للوصول إلى الملفات وقاعدة البيانات
3. قم بتشغيل `test.php` للتحقق من حالة النظام
4. تحقق من سجلات خطأ PHP إذا استمرت المشكلة

---

تم تطوير هذا التطبيق كمشروع للمساعدة في تحسين الصحة النفسية للمستخدمين من خلال تتبع المزاج وتقديم أنشطة وتقنيات لإدارة الإجهاد وتحسين الحالة المزاجية.
